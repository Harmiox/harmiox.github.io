/skins_fallback_v5 folder

This folder holds the fallback skin templates for skins using the evoSkin API version 5 (all skins made before b2evolution v6).

b2evolution Note: this folder only pertains to the public interface skins. The skins for the admin interface are to be found the the /skins_adm folder.

For example _feedback.php lists the comments and the trackbacks in a standard manner and can then be included in any skin. However, if time permits, the skin designer can also override _feedback.php in his customized skin folder.

This folder does not need to be accessed through the web.
